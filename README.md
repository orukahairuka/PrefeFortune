# PrefeFortune

## リンク
[GitHubリポジトリ](https://github.com/orukahairuka/PrefeFortune)

## 環境
- **Xcode**: 15.4
- **Swift**: 5.10

## プラットフォーム
- **iOS**: 17.5

## 実行手順
1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/orukahairuka/PrefeFortune.git
   ```
2. PrefeFortune.xcodeprojをXcodeで開きます。
3. 以下の手順でライブラリを追加します:
   - **File > Add Package Dependencies**を選択。
   - 以下のパッケージをそれぞれ追加してください:
     - Alamofire
     - Lottie
     - SwiftCSV
4. 実行するデバイスを選択します。
5. プロジェクトをビルド・実行します。
